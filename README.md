Riffy H
===

The missing pieces of nhentai

Requirements
------------

- [pnpm](https://pnpm.io/)
- Node LTS

Development
---

```
# Prebuild data
pnpm build:data

# Start development server
pnpm dev
```

Build
---

```
pnpm build
```
